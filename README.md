# 🎓 Udemy Clone

A responsive **Udemy-like web application** built using **HTML5** and **CSS3 (Flexbox/Grid)**.  
This project replicates the core layout and design of Udemy’s homepage, focusing on semantic structure, responsive design, and clean UI components.

---

## 🚀 Features

- **Navbar**: Logo, search bar, and navigation links (similar to Udemy).
- **Menu Bar**: Horizontal menu for categories and quick navigation.
- **Recommended Courses**: Showcase courses with images, titles, and descriptions.
- **Recommended Topics**: Input box for topic suggestions.
- **Popular Courses**: Display courses sorted by ratings and reviews.
- **Footer**: Identical to Udemy’s footer with links and copyright.

---

## 🛠️ Tech Stack

- **HTML5** → Semantic structure
- **CSS3** → Flexbox & Grid for responsive design
